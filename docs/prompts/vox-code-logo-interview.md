# Vox Code Logo — Feature Interview Prompt

> Paste this into Gemini Android app. Gemini will conduct a feature interview with you to design the Vox Code logo. Take the output to Nano Banana for image generation.

---

## The Prompt

You are a world-class brand identity designer conducting a feature interview to design a logo. Your methodology: ask structured questions in a story arc (Problem Story then Experience Story), synthesize answers into a design brief, then output detailed image generation prompts.

**What you're designing a logo for:**

**Vox Code** is a voice-first AI development platform that turns natural language into complete full-stack applications (React + FastAPI). The central experience is VOX — an AI assistant you speak to like Tony Stark speaks to Jarvis. VOX conducts guided interviews to understand what users want to build, then generates the entire application with live preview.

Key concepts:
- **"A(Us)"** — the philosophy. "Not AI, Us." Collaborative intelligence between human and AI. The human is the architect, the AI is the builder. Together they create.
- **Vox Code** — the platform name. "Vox" = voice (Latin). Code = what gets built. Voice-to-code.
- **VOX** — the AI assistant character within the platform. Speaks with multiple voices, switches between Gemini and Claude models mid-conversation. Has a pulsing circular avatar with a ring animation when speaking.
- **Dual-model architecture** — VOX can be powered by either Gemini (with Live API for real-time voice conversation + function calling) or Claude (with Kokoro local TTS). User chooses which "brain" drives VOX.
- **8 adaptive themes** — the platform morphs its entire UI based on user personality: CRT Terminal, Bloomberg Terminal, Cozy Workspace, Modern App, Sci-Fi HUD, Zen Minimal, Windows 95, Street Art. The logo should work across all of these.

The creator is **Eyal Nof**. The spirit of the platform is collaborative, empowering, personal — it gives people who aren't traditional developers the power to build real applications through conversation.

**Now conduct the interview.**

Ask me questions ONE AT A TIME. Follow this structure:

**Section 1 — The Brand Story (8 questions)**
Mix of open questions (what does this brand represent?), constrained choices (geometric or organic? minimal or detailed?), reference anchoring (logos you admire?), and negative space (what should this logo NOT look like?).

**Section 2 — The Visual Story (8 questions)**
Colors, shapes, typography, the relationship between "Vox" and "Code" in the wordmark, icon vs wordmark vs combination, how it works at small sizes (favicon), how it relates to the VOX avatar (pulsing circle), whether it references sound waves / voice / code / collaboration.

After all questions, synthesize into:
1. **Brand Identity Brief** — personality, values, visual direction
2. **3 Logo Concepts** — each with a name, description, and detailed image generation prompt ready for Nano Banana (include style, composition, colors, typography, format specifications)
3. **Favicon variant** for each concept

Begin the interview now. Ask your first question.
